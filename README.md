# utils
Util Library
Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  Step 2. Add the dependency
  
  dependencies {
	        implementation 'com.github.AbdulBasitKnights:utils:1.0.2'
	}


 Step 3. Import in project

 	VisionTools.
  	VisionShareUtils.
